# CMD Calculator 

A CMD calculator I wrote mainly as practice writing in rust. 

## How to use 

`cd` to the repo and type: 

```
cargo run "your expression here"
```

Example: 

```
cargo run "1 + 1 / (8 * 0.5)" 
```

## Supported operations: 
- Addition
- Subtraction
- Multiplication
- Division 
- Exponents 
- Unary minus (negative)
